The Up-Goer Five Text Editor (in JS)

CAN YOU EXPLAIN A HARD IDEA USING ONLY THE TEN HUNDRED MOST USED WORDS? IT'S NOT VERY EASY. TYPE IN THE BOX TO TRY IT OUT.

This tool was inspired by the THE UP-GOER FIVE TEXT EDITOR. It checks if all the words are part of the 1000 most common words (and their inflections). This version was created by Mathias Kahl. It's written entirely in very crappy JavaScript code, so just take a look at the source, if you want to know how it works.
